What is Notifyr?

Notifyr streamlines your application's communication needs by providing a robust backend service to relay critical messages. Effortlessly integrate SMS & Voice Calls using Twilio, and send Emails via standard SMTP, all managed through a single, secure interface.

Designed for developers and system administrators, Notifyr simplifies notification workflows, enhances security with Role-Based Access Control, and provides essential contact management features. Access the interactive API documentation to get started.

Core Features

Multi-Channel Relay

Consolidate SMS (Twilio), Voice Calls (Twilio), and Email (SMTP) notifications through one API.

RBAC Security

Implement granular permissions with built-in Role-Based Access Control for secure operations.

Contact Management

Maintain a centralized directory of contacts and their preferred notification channels.

Twilio Integration

Reliable SMS and Voice capabilities powered by the industry-leading Twilio API.

Flexible SMTP

Utilize standard SMTP protocols to send emails via your preferred provider or server.

Developer Focused

Core Technologies

Notifyr leverages robust and widely-adopted technologies:

  • Twilio API: Powering programmable SMS & Voice functionalities.
  • SMTP Protocol: Standard email relay for maximum compatibility.
  • Secure Backend Framework: Built upon FastAPI (Python) with security best practices.
  • Database: Securely stores configuration and contact data (e.g., PostgreSQL, MySQL).
  • RBAC Implementation: Custom or library-based Role-Based Access Control logic.
